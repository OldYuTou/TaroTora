===============================================
TaroTora 远程控制服务端 v1.0
===============================================

【快速启动】

方法1：双击 "双击启动.bat" 运行交互式配置器

方法2：运行 "启动服务端.bat" 启动完整配置流程


【系统要求】

- Windows 7/10/11
- Node.js 18+ （必需）
  下载地址：https://nodejs.org/


【首次使用步骤】

1. 安装 Node.js（如未安装）

2. 双击 "双击启动.bat"

3. 按照提示输入配置：

   服务端口 [3000]:
   - 保持默认或输入其他端口（如 3001、8080）

   访问令牌 [随机生成]:
   - 建议设置复杂密码，客户端连接时需要

   启用 FRP 内网穿透 [y/N]:
   - 不需要外网访问则输入 N
   - 需要则输入 Y，然后输入 FRP 服务器信息

4. 确认后立即启动服务器

5. 记住以下信息用于客户端连接：
   - IP 地址: 电脑的局域网 IP
   - 端口号: 你设置的端口（默认 3000）
   - 访问令牌: 你设置的密码


【配置说明】

配置文件: .env
- 首次运行自动生成
- 可随时修改后重启生效


【常见问题】

Q: 提示未检测到 Node.js
A: 访问 https://nodejs.org/ 下载并安装

Q: 端口被占用
A: 选择其他端口，如 3001、8080、8888

Q: 如何修改配置
A: 删除 .env 文件后重新运行启动器

Q: 如何启用外网访问
A: 需要自备 FRP 服务器，启动时选择启用 FRP


【文件说明】

双击启动.bat       - 快速启动入口
启动服务端.bat     - 完整启动脚本
launcher.js        - 启动器核心程序
src/               - 服务器程序
frp/               - FRP 客户端（可选）
.env               - 配置文件（自动生成）
