# STL 文件预览网站使用指南

一个基于 Web 的 STL 3D 模型预览工具，支持自动扫描目录、下拉选择切换、默认加载最新文件。

## 功能特点

- 🔍 **自动扫描** - 自动检测 `stl/` 目录下的所有 `.stl` 文件
- 📂 **下拉选择** - 通过下拉框快速切换不同模型
- ⚡ **智能默认** - 自动加载修改时间最新的文件
- 🎮 **交互操作** - 鼠标控制：左键旋转、右键平移、滚轮缩放
- 📊 **模型信息** - 显示文件名、大小、顶点数等详细信息
- 📱 **响应式设计** - 支持桌面和移动设备

---

## 快速开始

### 1. 安装依赖

```bash
cd stl-viewer
npm install
```

### 2. 启动服务器

```bash
npm start
```

看到以下输出表示启动成功：
```
STL Viewer 服务器运行在 http://localhost:3000
STL 文件目录: /path/to/stl-viewer/stl
```

### 3. 访问网站

打开浏览器访问：http://localhost:3000

---

## 添加自己的 STL 文件

### 方法一：直接复制

将你的 `.stl` 文件复制到项目的 `stl/` 目录：

```bash
cp /path/to/your/model.stl ./stl/
```

刷新浏览器页面即可看到新文件。

### 方法二：通过文件管理器

1. 打开 `stl-viewer/stl/` 文件夹
2. 将 STL 文件拖入该目录
3. 刷新网页

## 界面说明

### 顶部控制栏

```
🔧 STL 文件预览器
选择文件: [cube.stl (1.44 KB) [最新]]  共 2 个文件
```

- **下拉框** - 列出所有可用的 STL 文件，按修改时间倒序排列
- **文件数量** - 显示当前检测到的文件总数
- **"[最新]"标签** - 表示该文件是最近修改的，页面加载时会默认选中

### 3D 视图区域

- **加载动画** - 加载模型时显示旋转的加载指示器
- **空状态提示** - 没有 STL 文件时显示引导信息
- **错误提示** - 文件加载失败时显示错误信息

### 模型信息面板

加载模型后显示：

| 项目 | 说明 |
|------|------|
| 文件名 | 当前显示的 STL 文件名 |
| 文件大小 | 格式化后的文件大小（Bytes/KB/MB） |
| 修改时间 | 文件最后修改时间 |
| 顶点数 | 3D 模型的顶点数量 |

## 切换模型

### 通过下拉框切换

1. 点击顶部的下拉选择框
2. 从列表中选择想要查看的文件
3. 页面自动加载并显示选中的 3D 模型

### 自动加载最新文件

每次刷新页面时，系统会自动：
1. 扫描 `stl/` 目录
2. 按修改时间排序
3. 自动加载最新的文件

## 支持的文件格式

- **格式**: ASCII STL 或 Binary STL
- **扩展名**: `.stl`（不区分大小写）
- **编码**: 支持标准 STL 格式

## 常见问题

### Q: 页面显示"暂无 STL 文件"

**原因**: `stl/` 目录下没有 `.stl` 文件  
**解决**: 将 STL 文件放入 `stl/` 目录后刷新页面

### Q: 文件加载失败

**原因**: 
- 文件格式不是标准 STL
- 文件损坏
- 文件过大导致内存不足

**解决**:
- 使用 3D 建模软件重新导出 STL
- 尝试简化模型（减少面数）
- 检查文件是否能在其他软件中正常打开

### Q: 3D 模型显示不正常

**原因**: 模型尺寸过大或过小，或坐标原点不在中心

**解决**: 
- 使用滚轮缩放调整视图
- 在 3D 建模软件中将模型居中到原点
- 调整模型尺寸到合适范围

### Q: 如何修改端口号？

**解决**: 修改 `server.js` 中的 `PORT` 变量：

```javascript
const PORT = process.env.PORT || 3000;  // 改为其他端口，如 8080
```

或使用环境变量：

```bash
PORT=8080 npm start
```

## 技术栈

- **后端**: Node.js + Express
- **前端**: HTML5 + CSS3 + JavaScript (ES6+)
- **3D 渲染**: Three.js r128
- **STL 加载**: Three.js STLLoader
- **交互控制**: Three.js OrbitControls

## 自定义配置

### 修改 STL 文件目录

编辑 `server.js`，修改 `STL_DIR`：

```javascript
const STL_DIR = path.join(__dirname, 'your-stl-folder');
```

### 修改默认颜色和样式

编辑 `public/app.js`：

```javascript
// 模型颜色
const material = new THREE.MeshPhongMaterial({
  color: 0x00d4ff,  // 改为其他颜色，如 0xff0000 (红色)
  specular: 0x111111,
  shininess: 200
});
```

### 修改背景颜色

编辑 `public/app.js`：

```javascript
scene.background = new THREE.Color(0x0f0f23);  // 改为其他颜色
```
